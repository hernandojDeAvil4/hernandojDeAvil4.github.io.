Solución por computadora de la expansión en fracciones parciales

Introducción

Existen muchos lenguajes de cómputo de alto nivel que pueden ser utilizados para encontrar la solución de la transformada inversa de Laplace, MATLAB es el más popular de estos. No obstante, también se puede implementar Python para encontrar la solución. En esta sección veremos cómo se puede instalar la librería Sympy para trabajar con soluciones simbólicas dentro de Python y como solucionar la transformada inversa de Laplace. 

Instalación de Python:

Primero debemos descargar Python, esto lo podemos hacer directamente desde la página oficial (asegúrese de descargarlo de acuerdo a su sistema operativo, en el caso de Windows se puede descargar desde Microsoft Store, pero esta versión suele estar desactualizada). Luego de instalar Python se recomienda instalar una interface de usuario, la instalación ya debe tener una, pero no suele ser muy cómodo realizar depuración de código con este. Entre algunos interfaces que recomiendan los usuarios se encuentra:

  1. PyQt5
  2. wxPython
  3. PyGUI
  4. Visual Estudio Code
Otra opción muy recomendada es instalar Conda el cual tiene muchas librerías predeterminadas.

Instalación de Sympy

Se puede instalar la librería Sympy de dos formas:

  • Descargar el archivo comprimido de su pagina oficial
  • Instalar Sympy en el sistema con el comando pip

Para instalarlo mediante el comando pip solo se necesita introducir los siguientes comandos en command prompt windows o terminal Linux:

  1. python -m pip install --upgrade pip
  2. pip3 install sympy
En este punto ya debería tener instalada la librería si por alguna razón acontece un problema con el comando pip recomiendo visitar How To Install Pip On Windows
